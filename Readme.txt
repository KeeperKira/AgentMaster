Данный проект является редактором Агента для нейронных сетей.

представляет из себя упращённую версию таких проектов как n8n

его функционал это маршрутизация запроса с возможностью добавлять текст к запросу.

само рабочее пространстов разделено на 2 поля
1 каталог инструментов(с лева в виде столбца)
2 рабочее поле. куда претягиваются инструменты из 1го поля.

сами инструменты имеют точки входа и выхода. которые пользователь может объединять
протянув точку входа к точке выхода (к одному выходу сколько угодно входов/ к одному входу только один выход)

	с точки зрения UI есть шарики на инструменте(квадрате) вход/выход которые можно соединять между собой

инструменты
1. текст	порты {0 входов, 1 выход} есть 2 поля для ввода{1 Name, 2 Text} 
2. ввод		порты {0 входов, 1 выход} обязательное поле, всегда есть на рабочем поле. имеет только 1 выход. представляет из себя точку откуда будут приходить запросы от пользователя(пользователя агентом)
3. триплет	порты {3 входа, 1 выход} 1 поле {1 Name}
4. роутер	порты {1 вход, 1 выход} 3 поля для ввода{ 1 Name, 2 URL, 2 API_KEY}
5. вывод	порты {1 вход, 1 выход} также является обязательным на рабочем пространстве.

примеры работы

1.на поле 3 инструмента (ввод(1),роутер(2),вывод(3)) обеденив 1->2->3 получаю чат с ИИ без сохранения контекста.
добавив на поле триплет(4) подключив 1->4->2->3->4 получаю чат с ИИ с сохранением контекста.


сам функционал будет реализован в другой программе, а резутатом данной программы должен быть json файл с помощю которого можно загрузить агента в программу "исполнитель"
тоесть в файле должно быть сохранено 
1.инструмены
2.связи
3.тексты полей.
